# Concept A - Adaptive Core Launch

## Description
Rebuild TriTrainer around a deterministic planning engine that is structurally ready for adaptive logic, while shipping core goals and PR tracking in a new modular UI.

## Pros
- Balances speed and architectural safety.
- Keeps hard constraints intact (C# backend, Aspire local orchestration).
- Creates foundation for advanced adaptation without rework.

## Cons
- Early releases may feel less intelligent than full adaptive vision.
- Requires careful messaging so users understand roadmap.

## Estimated Effort
- Sprint 1: domain expansion, baseline UI redesign, migration safety.
- Sprint 2: adaptive plan adjustments and deeper insights.
